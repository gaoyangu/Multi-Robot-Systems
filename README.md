## 89-689 Multi-Robot Systems

## Usage
### 1. Simulating multiple robots in Stage
```bash
$ roslaunch stage_multi stage_multi.launch
```

### 2. Robots Synchronization
```bash
$ rosrun multi_sync monitor 4
$ roslaunch stage_multi stage_multi.launch
```
